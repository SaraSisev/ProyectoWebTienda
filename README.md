# ProyectoWebTienda 

## Descripción  
Proyecto de página web para una tienda de compra/venta de LEGOs. Construida con **HTML**, **CSS** y **JavaScript**. El objetivo es ofrecer una interfaz sencilla y funcional donde los usuarios puedan ver productos, agregar al carrito, y (en el futuro) realizar órdenes.
